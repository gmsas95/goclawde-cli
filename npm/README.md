# Myrai - npm Package

This is the npm wrapper for [Myrai](https://github.com/gmsas95/goclawde-cli), your personal AI assistant.

## Installation

```bash
# Install globally
npm install -g myrai

# Or use with npx (no installation required)
npx myrai --help
```

## Usage

```bash
# First time setup
myrai onboard

# Start interactive CLI
myrai --cli

# Start server
myrai server

# Get help
myrai --help
```
